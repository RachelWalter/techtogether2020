About the Project

This project is a submission to TechTogether Boston, a gender-focused hackathon that aims to empower women through technology. I wanted to provide resources to survivors of sexual violence, as well as provide a review and visualization of public data on sexual violence and provide an alternative system of dealing with the aftermath of sexual violence through a beginner's guide to Restoratrive Justice.

The source code of this website and my data analysis is available on my GitHub. My data science tools included Jupyter Notebooks and Python (especially the pandas, matplotlib, and folium libraries). I coded the website using HTML, CSS, and some Javascript. Some features of this site are widgets created by other organizations focused on sexual violence; these widgets are credited accordingly. The iconography in the Restorative Justice Approach section are my original works created with Adobe Photoshop.
